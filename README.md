# Pharmacy-Manegment-System
MedStock Pro is a desktop application I built using Qt C++ and SQLite to help small pharmacies manage their daily operations — from inventory to sales — in one clean, simple interface.

This is a desktop app for small pharmacies to manage inventory, track sales, and get alerts on low stock or expiring medicines — all from one clean interface.

## Features

- **Login / Signup** – with duplicate username checks  
- **Add Medicines** – name, formula, batch, quantity, price, supplier, expiry  
- **Sell Medicines** – stock updates automatically  
- **Search** – by name or chemical formula  
- **Sales Reports** – today, yesterday, last 7 or 30 days  
- **Low Stock Alerts** – medicines below 10 units  
- **Expiry Alerts** – with color-coded warnings  

---

## Tech Stack

- **Frontend**: Qt C++ 
- **Backend**: SQLite  
- **IDE**: Visual Studio 2022 / Qt Creator  
- **Language**: C++17

## How to Run
Before running the project install the following 

* Visual Studio 2022 with Desktop development with C++
* Qt 6
* Qt Visual Studio Tools extension 
* Git
---

## 1. Clone the Repository

```bash
git clone https://github.com/mhuzaifalaghari-sketch/Pharmacy-Manegment-System.git
cd Pharmacy-Manegment-System
```

## 2. Open the Project
### Visual Studio 2022

1. Open the .sln file.
2. Make sure Qt Visual Studio Tools is installed.
3. Ensure the correct Qt version is selected.


## 3. Build the Project

Build the project using:

* Build → Build Solution
---
## Database

The application uses SQLite.

On the first run, the SQLite database will be created automatically if it does not already exist.
